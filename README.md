Ce projet a permis de mettre en pratique les concepts fondamentaux des applications réparties grâce à la technologie Java RMI. L'implémentation d'un service de conversion de devises a démontré :
 
 Réalisations techniques:
Architecture client-serveur avec séparation claire des responsabilités

Communication distante transparente via l'invocation de méthodes à distance

Découplage grâce à l'interface IConversion définissant le contrat de service

Gestion du registre RMI pour la publication et la découverte de services

 Compétences acquises
Maîtrise du workflow RMI (registry, stub, skeleton)

Gestion de la sérialisation des objets pour les appels distants

Configuration des politiques de sécurité et codebase

Débogage d'applications distribuées
